
# Car Shwo!

- The Car Shwo project is a platform where you can find various types of cars.
    - User Features
        - Login
        - Registration
        - Email sending
        - Browse cars
        - Discover car details
        - Schedule appointments
    - Admin Features
        - View all cars scheduled by users
        - View sold cars
        - Edit or delete cars
        - View the dashboard

- Technologies Used
    - Front End: NextJS, TypeScript, Zod
    - Back End: NestJS
    - Database: PostgreSQL
    - Tailwind CSS
    - TypeScript

## Install Dependencies
$ npm install

## Start the Application
$ npm run dev
